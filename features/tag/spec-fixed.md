# 태그 기능 정의서

### 추가 기능
- 노트에 태그를 추가하고 관리할 수 있다.

### 기능 요구사항
- 노트에 태그를 추가할 수 있다.
- 태그 우측 x버튼 클릭하여 삭제할 수 있어야 한다.
- 태그 목록을 노트 상세에서 확인, 수정할 수 있어야 한다.
- 목록(NoteList)에서 나열된 태그 목록을 볼 수 있어야 한다.

### UI 상세

| 항목 | 결정 |
|------|------|
| 입력 방식 | 태그 목록 끝 `+` 버튼 클릭 → 인풋이 포함된 태그 뱃지 생성 |
| 입력 위치 | 태그 뱃지 안에 input 인라인 렌더링 (NoteEditor 내) |
| 목록 표시 | `flex-row`로 태그 뱃지 나열 (NoteList의 NoteItem에 표시) |
| 수정 방식 | 태그 클릭 시 인라인 편집 (뱃지 → input 전환) |
| 중복 처리 | Toast 표시: "이미 존재하는 태그입니다." |

### 비고
- 태그는 문자열로 저장한다
- 태그 최대 10자까지
- 중복은 불허
