함수 명세

함수 메서드 엔드포인트 입력 타입 반환 타입 비고
fetchNotes GET /notes — Promise<Note[]> 전체 목록 조회
createNote POST /notes Omit<Note, 'id'|'createdAt'|'updatedAt'> Promise<Note> createdAt/updatedAt 클라이언트 생성
updateNote PATCH /notes/:id id: string, Partial<Note> Promise<Note> updatedAt 자동 갱신
deleteNote DELETE /notes/:id id: string Promise<void> 응답 바디 없음

⚠️ 현재 한계

• 재시도(retry) 없음
• AbortController(요청 취소) 없음
• e.message 접근 시 타입 보장 없음 (e가 Error 객체가 아닐 수 있음)
• alert() 사용 — 향후 토스트 UI로 교체 예정
